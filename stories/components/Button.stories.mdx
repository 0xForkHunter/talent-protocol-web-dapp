# Buttons

Custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

## Examples

There are two currently supported types, `primary` and `secondary`. We use react-bootstrap button underneath so we accept any prop they use.

export const Template = (args) => <Button {...args} />
