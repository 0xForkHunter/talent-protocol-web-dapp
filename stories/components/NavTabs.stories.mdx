# NavTabs

Custom style for navigational tabs.

## Examples

This example is extracted from the talent screen. We make use of `const url = new URL(document.location)` for figuring out the pathname and checking against the tab url, therefore we don't have the active tab demoed in storybook because we can't make use of different routes.

export const Template = (args) => <NavTabs {...args} />

<Canvas isColumn>
  <Story name="default" args={{ 
      tabs: [
        { name: "All", url: "/talent" },
        { name: "Active", url: "/talent/active" },
        { name: "Upcoming", url: "talent/upcoming" }
      ],
      sortOptions: [
        { label: "Newest", param: "newest" },
        { label: "Market Cap", param: "market_cap" },
        { label: "Activity", param: "activity" }
      ]
    }}>
    {Template.bind({})}
  </Story>
</Canvas>
